# Animated Skills Bar UI Design

Um componente de visualização de competências (Skills Bar) moderno, elegante e totalmente responsivo desenvolvido exclusivamente com HTML5 e CSS3. O projeto apresenta uma seção estruturada de barras de progresso dinâmicas que indicam níveis de proficiência em diferentes tecnologias através de animações fluidas de preenchimento e balões de porcentagem (*tooltips*).

## Tecnologias Utilizadas

* HTML5: Estruturação semântica da lista de competências, agrupando rótulos de texto e contêineres de progresso.
* CSS3: Estilização completa, uso de cantos suavizados em estilo pílula, gerenciamento de profundidade com sombras e paleta baseada em tons de roxo.
* Animações em CSS: Utilização de keyframes (`@keyframes`) nativos para controlar o preenchimento progressivo das barras de forma otimizada para o navegador.

## Funcionalidades e Técnicas Aplicadas

* Animação de Preenchimento Dinâmico: Aplicação de efeitos de expansão linear contínuos que fazem com que a linha roxa de progresso cresça horizontalmente do 0% até o valor final determinado ao carregar a página.
* Balões de Porcentagem Customizados (Tooltips): Inclusão de pequenos blocos pretos flutuantes contendo as métricas de proficiência (90%, 60%, 85%, 50%, 75%). Estes balões contam com um detalhe em formato de seta na base e deslocam-se de forma sincronizada com o topo de cada barra.
* Layout Organizado e Limpo: Uso de uma estrutura empilhada onde cada tecnologia (HTML, CSS, jQuery, Python, MySQL) possui um rótulo bem posicionado acima de seu respectivo trilho de progresso fosco cinza-claro.
* Design Estilo Card Flutuante: O contêiner principal branco apresenta cantos levemente arredondados por meio de `border-radius` e uma sombra projetada (`box-shadow`) sutil que confere profundidade tridimensional sobre o plano de fundo roxo plano.
